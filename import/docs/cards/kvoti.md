# Kvóti-kort - Efnahagskort (frá 1984)

Kort sem tengjast kvótakerfinu og efnahag sjávarútvegs.

## Grunnatburðir

### Kvótakerfið tekur gildi (1984)
```
┌─────────────────────────────────────────┐
│  ⚖️ KVÓTI: Kvótakerfið lögleitt        │
│                                         │
│  Aflareynsla síðustu 3 ára ákvarðar     │
│  kvóta þinn.                            │
│                                         │
│  ÁHRIF:                                 │
│  • Kvóti reiknaður út frá sögu          │
│  • Nýir leikmenn fá LÍTINN kvóta        │
│  • Reynslumiklir fá STÓRAN kvóta        │
│                                         │
│  ⚠️ Héðan í frá gildir kvótaregla       │
└─────────────────────────────────────────┘
```

│                                         │

```

## Kvótareglugerðir

### Skipstjórakvóti
```
┌─────────────────────────────────────────┐
│  ⚖️ KVÓTI: Skipstjórakvóti             │
│                                         │
│  Sérregla: Reynsla skipstjóra gildir,   │
│  ekki skipsins.                         │
│                                         │
│  ÁHRIF:                                 │
│  • Skipstjóri: +20% kvóti               │
│  • Nýr skipstjóri = minna               │
│  • Gamall skipstjóri: Getur leigt út    │
└─────────────────────────────────────────┘
```

### Kvótaþak
```
┌─────────────────────────────────────────┐
│  ⚖️ KVÓTI: Kvótaþak sett               │
│                                         │
│  Hámark 12% hjá einu fyrirtæki.         │
│                                         │
│  ÁHRIF:                                 │
│  • Ef þú átt >12%: Verður að selja      │
│  • Stórar útgerðir: Skipta í dótturfél. │
│  • Nýir möguleikar fyrir smærri         │
└─────────────────────────────────────────┘
```

### Byggðakvóti

┌─────────────────────────────────────────┐
│  ⚖️ KVÓTI: Byggðakvóti úthlutaður      │
│                                         │

```
